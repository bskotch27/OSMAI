# OSMAI
To study the impact of AI tools on Crowdsourcing

## View the Website

The project includes a static HTML page that you can view in several ways:

### Option 1: Open Directly in Browser
Simply open `index.html` in your web browser by double-clicking the file or dragging it into your browser window.

### Option 2: Local Development Server
Run a local web server to view the page:

```bash
# Using Python 3
python3 -m http.server 8080

# Using Python 2
python -m SimpleHTTPServer 8080

# Using Node.js (if you have http-server installed)
npx http-server -p 8080
```

Then visit `http://localhost:8080` in your browser.

### Option 3: GitHub Pages
To host this website publicly using GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch you want to deploy (e.g., `main`)
4. Click "Save"
5. Your site will be published at `https://bskotch27.github.io/OSMAI/`

## About
This project explores how artificial intelligence tools are transforming crowdsourcing practices and workflows.
